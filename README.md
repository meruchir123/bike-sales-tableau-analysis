# 🚴 Bike Store Data Analysis (Tableau Dashboard)

## 📊 Project Overview

This project presents an interactive Tableau dashboard analyzing bike store sales data across multiple countries.
It focuses on **sales distribution, shipment analysis, and salesperson performance**.

---

## 📌 Dashboard Highlights

### 🌍 Country-wise Sales Analysis

* Visual comparison of total sales across countries:

  * Australia
  * Canada
  * India
  * New Zealand
  * UK
  * USA
* Helps identify **top-performing regions**

---

### 📦 Country vs Boxes Shipped

* Horizontal bar chart showing shipment volume by country
* Highlights:

  * Australia has the highest shipment volume (~30K boxes)
* Useful for understanding **logistics and demand distribution**

---

### 👨‍💼 Salesperson vs Sales

* Bubble chart representing sales contribution of each salesperson
* Each bubble = individual salesperson
* Helps identify:

  * High performers
  * Sales distribution among team members

---

### 🎛️ Interactive Filter

* **Boxes Shipped Range Filter** (26,580 – 32,647)
* Allows dynamic filtering of shipment-related insights

---

## 🛠️ Tools Used

* Tableau Public (Dashboard Creation & Visualization)
* CSV Dataset (Bike Store Data)

---

## 📂 Dataset Features

* Country
* Sales Amount
* Boxes Shipped
* Salesperson Data

---


## 📈 Key Insights

* Australia shows the highest shipment volume
* Sales are relatively evenly distributed
